# Лабораторная работа №2: Мини-оболочка с файловыми командами

## Введение
Данная лабораторная работа представляет собой реализацию мини-оболочки на Python
с поддержкой основных файловых операций.

В папке src находится основной код моей программы.
В папке commands реализованы следующие команды:

ls - отображение файлов и каталогов

cd - переход в указанный каталог (поддержка .. и ~)

cat - вывод содержимого файла

cp  - копирование файлов и каталогов

mv  - перемещение и переименование

rm - удаление файлов и каталогов


Папка minishell это по сути один из главных элементов
нашей программы. Класс Minishell организует всю логику работы оболочки.
В папке tests находятся тесты, которые помогают оценить работоспособность кода
(мы проверяет корректрность работы не только для верно введенных выражений, но и 
для ошибочных, чтобы убедиться в том, что 
наша программа верно предупреждает пользователя о неверном вводе)

Для запуска у вас должен быть установлен Python (точно работает на 3.13).
Нужно перейти в файл main и запустить программу. Вас встретит небольшая инструкция. 
После чего можете вводить желаемое выражение
